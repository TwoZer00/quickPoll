# QuickPoll

Web app to create polls updated in real time.

## Tech Stack

- [React](https://react.dev/) (JavaScript)
- [Vite](https://vitejs.dev/) - Build tool
- [Firebase](https://firebase.google.com/) - Firestore, Analytics, Anonymous Auth
- [Material UI](https://mui.com/material-ui/) - UI components
- [React Router DOM](https://reactrouter.com/en/main) - Routing
- [Recharts](https://recharts.org/) - Charts
- [React Spring](https://www.react-spring.dev/) - Animations
- [dayjs](https://day.js.org/) - Date handling
- [Netlify](https://www.netlify.com/) - Deployment

## Getting Started

Requires Node.js >= 22.

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
